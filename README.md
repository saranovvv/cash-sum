# cash-sum
Projekt aplikacji mobilnej kasy dla sklepów lub firm

Nie każdy ze sklepów posiada kasę fiskalną bądź chce zarządzać swoją sprzedażą przez zeszyt. 
Z aplikacją Cash&Sum wszytkie czynności zarządzania sprzedażą mogą być wykonywane na twoim własnym laptopie lub komputerze. 

Z czasem przyjdzie moment na aplikacje mobilne, lecz teraz skupmy się na podstawach.
